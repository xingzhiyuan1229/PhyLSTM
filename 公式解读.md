# 公式解读：Physics-Informed Multi-LSTM Networks for Metamodeling of Nonlinear Structures

## 第 2 章：面向元建模的物理信息多 LSTM 网络

### 2.1 结构动力方程与状态空间描述

#### 公式 (1)——非线性运动方程（Equation of Motion, EOM）

\[
\mathbf{M}\ddot{\mathbf{u}} +\underbrace{\mathbf{C}\dot{\mathbf{u}} + \lambda\mathbf{K}\mathbf{u} + (1 - \lambda)\mathbf{K}\mathbf{r}}_{\mathbf{h}} = -\mathbf{M}\mathbf{\Gamma} a_{g}
\tag{1}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\mathbf{M}\) | 质量矩阵（Mass matrix），对角正定矩阵，各元素为楼层质量 |
| \(\mathbf{C}\) | 阻尼矩阵（Damping matrix），通常采用瑞利阻尼 |
| \(\mathbf{K}\) | 刚度矩阵（Stiffness matrix） |
| \(\mathbf{u},\dot{\mathbf{u}},\ddot{\mathbf{u}}\) | 相对于地面的位移、速度、加速度向量 |
| \(\mathbf{r}\) | 滞回位移参数（Hysteretic displacement），辅助非可观测量，用于描述材料的滞回行为 |
| \(\lambda \in (0,1]\) | 屈服后刚度与弹性刚度之比（Post-yield to pre-yield stiffness ratio） |
| \(a_{g}\) | 地面加速度时程（Ground acceleration） |
| \(\mathbf{\Gamma}\) | 力分布向量（Force distribution vector），通常为全1列向量（均匀地震输入） |
| \(\mathbf{h}\) | 总非线性恢复力（Total nonlinear restoring force） |

**详细推导与解释：**

公式 (1) 是结构动力学的核心方程，其物理来源是牛顿第二定律：

\[
\text{惯性力} + \text{阻尼力} + \text{恢复力} = \text{外部激励}
\]

具体而言，恢复力 \(\mathbf{h}\) 被分解为两部分：
- 弹性部分：\(\lambda\mathbf{K}\mathbf{u}\)，对应屈服后仍保留的弹性刚度贡献；
- 滞回部分：\((1-\lambda)\mathbf{K}\mathbf{r}\)，对应由材料非线性引入的滞回位移贡献。

当 \(\lambda = 1\) 时，退化为线弹性系统（无滞回）：

\[
\mathbf{M}\ddot{\mathbf{u}} + \mathbf{C}\dot{\mathbf{u}} + \mathbf{K}\mathbf{u} = -\mathbf{M}\mathbf{\Gamma} a_{g}
\]

当 \(\lambda < 1\) 时，系统具有非线性滞回特性，此时 \(\mathbf{r}\) 是一个随时间演化的内部隐藏变量，不可通过传感器直接测量，这正是本文建立物理信息网络的核心动机之一。

右端项 \(-\mathbf{M}\mathbf{\Gamma} a_{g}\) 是地震惯性激励力，负号表示地震力方向与地面加速度方向相反（相对运动原理）。

---

#### 公式 (2)——质量归一化后的运动方程

\[
\ddot{\mathbf{u}} + \mathbf{g} = -\mathbf{\Gamma} a_{g}
\tag{2}
\]

**推导过程：**

对公式 (1) 两端同时左乘 \(\mathbf{M}^{-1}\)（质量矩阵逆）：

\[
\mathbf{M}^{-1}\mathbf{M}\ddot{\mathbf{u}} + \mathbf{M}^{-1}\mathbf{h} = -\mathbf{M}^{-1}\mathbf{M}\mathbf{\Gamma} a_{g}
\]

\[
\ddot{\mathbf{u}} + \underbrace{\mathbf{M}^{-1}\mathbf{h}}_{\mathbf{g}} = -\mathbf{\Gamma} a_{g}
\]

其中定义**质量归一化恢复力**：

\[
\mathbf{g}(t) = \mathbf{M}^{-1}\mathbf{h}(t)
\]

\(\mathbf{g}(t)\) 是一个未知的非线性隐函数，其完整表达式为：

\[
\mathbf{g}(t) = \mathcal{G}(\mathbf{Z}(t))
\]

其中 \(\mathcal{G}\) 是待学习的未知函数，\(\mathbf{Z} = \{\mathbf{z}_1, \mathbf{z}_2, \mathbf{z}_3\}^T = \{\mathbf{u}, \dot{\mathbf{u}}, \mathbf{r}\}^T\) 是系统的**状态空间变量**。

**公式 (2) 与公式 (1) 的联系：**

公式 (2) 是公式 (1) 经过质量归一化后的通用形式。质量归一化具有两个优点：
1. 消除了质量矩阵的显式影响，使方程更通用；
2. 将非线性恢复力统一表示为 \(\mathbf{g}\)，便于神经网络学习这一未知的隐函数关系。

公式 (2) 是整个物理约束损失函数构建的起点。在网络训练中，要求网络的预测满足：

\[
\ddot{\mathbf{u}} + \mathbf{g} + \mathbf{\Gamma} a_{g} \rightarrow 0
\]

这构成了**运动方程物理约束**（EOM constraint），也是后续公式 (10) 损失函数 \(\mathcal{J}_e\) 的直接来源。

### 2.2 LSTM 网络基础架构

#### 公式 (3)——遗忘门（Forget Gate）

\[
\mathbf{f}_{t}^{(l)} = \sigma \left(\mathbf{W}_{x_f}^{(l)}\mathbf{x}_{t}^{(l)} + \mathbf{W}_{h_f}^{(l)}\mathbf{h}_{t-1}^{(l)} + \mathbf{b}_{f}^{(l)}\right)
\tag{3}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\mathbf{f}_{t}^{(l)}\) | 第 \(l\) 层在时间步 \(t\) 的遗忘门输出，取值范围 \([0,1]\) |
| \(\mathbf{W}_{x_f}^{(l)}\) | 遗忘门中输入 \(\mathbf{x}_t\) 对应的权重矩阵 |
| \(\mathbf{W}_{h_f}^{(l)}\) | 遗忘门中隐状态 \(\mathbf{h}_{t-1}\) 对应的权重矩阵 |
| \(\mathbf{b}_{f}^{(l)}\) | 遗忘门偏置向量 |
| \(\sigma(\cdot)\) | Sigmoid 激活函数：\(\sigma(x) = \frac{1}{1+e^{-x}} \in (0,1)\) |

**详细解释：**

遗忘门决定**上一时刻的细胞状态 \(\mathbf{c}_{t-1}^{(l)}\) 有多少比例被保留**。当 \(\mathbf{f}_t^{(l)} \approx 1\) 时，几乎全部保留；当 \(\mathbf{f}_t^{(l)} \approx 0\) 时，几乎全部遗忘。这使得 LSTM 能够自适应地管理长期记忆，避免传统 RNN 中的梯度消失问题。

Sigmoid 函数的选择保证了遗忘门输出在 \([0,1]\) 范围内，物理上对应"保留比例"的语义。

---

#### 公式 (4)——输入门（Input Gate）

\[
\mathbf{i}_{t}^{(l)} = \sigma \left(\mathbf{W}_{x_i}^{(l)}\mathbf{x}_{t}^{(l)} + \mathbf{W}_{h_i}^{(l)}\mathbf{h}_{t-1}^{(l)} + \mathbf{b}_{i}^{(l)}\right)
\tag{4}
\]

**详细解释：**

输入门决定**当前时刻的候选细胞状态 \(\tilde{\mathbf{c}}_t^{(l)}\) 有多少比例被写入细胞记忆**。与遗忘门类似，其输出也在 \([0,1]\) 范围内，控制新信息的流入量。

---

#### 公式 (5)——候选细胞状态（Candidate Cell State）

\[
\tilde{\mathbf{c}}_{t}^{(l)} = \tanh \left(\mathbf{W}_{x_c}^{(l)}\mathbf{x}_{t}^{(l)} + \mathbf{W}_{h_c}^{(l)}\mathbf{h}_{t-1}^{(l)} + \mathbf{b}_{c}^{(l)}\right)
\tag{5}
\]

**详细解释：**

候选细胞状态是由当前输入 \(\mathbf{x}_t^{(l)}\) 和上一步隐状态 \(\mathbf{h}_{t-1}^{(l)}\) 共同生成的**新候选记忆**，取值范围为 \([-1,1]\)（tanh 函数的值域）。它代表了在当前时间步"可能需要记住"的信息。

tanh 函数的选择使候选值具有正负两个方向，相比 Sigmoid 具有更好的梯度特性和表达能力。

---

#### 公式 (6)——输出门（Output Gate）

\[
\mathbf{o}_{t}^{(l)} = \sigma \left(\mathbf{W}_{x_o}^{(l)}\mathbf{x}_{t}^{(l)} + \mathbf{W}_{h_o}^{(l)}\mathbf{h}_{t-1}^{(l)} + \mathbf{b}_{o}^{(l)}\right)
\tag{6}
\]

**详细解释：**

输出门决定**当前细胞状态 \(\mathbf{c}_t^{(l)}\) 有多少比例作为隐状态输出**。它控制了细胞内部记忆如何暴露给下一层或输出层。

---

#### 公式 (7)——细胞状态更新（Cell State Update）

\[
\mathbf{c}_{t}^{(l)} = \mathbf{f}_{t}^{(l)} \odot \mathbf{c}_{t-1}^{(l)} + \mathbf{i}_{t}^{(l)} \odot \tilde{\mathbf{c}}_{t}^{(l)}
\tag{7}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\odot\) | 哈达玛积（Hadamard product），即逐元素乘法 |

**推导与解释：**

细胞状态更新是 LSTM 的核心机制，分为两个步骤：

**步骤 1——选择性遗忘旧记忆：**

\[
\mathbf{f}_{t}^{(l)} \odot \mathbf{c}_{t-1}^{(l)}
\]

遗忘门 \(\mathbf{f}_t^{(l)}\) 逐元素乘以上一步细胞状态 \(\mathbf{c}_{t-1}^{(l)}\)，保留需要保留的部分，遗忘需要遗忘的部分。

**步骤 2——选择性写入新记忆：**

\[
\mathbf{i}_{t}^{(l)} \odot \tilde{\mathbf{c}}_{t}^{(l)}
\]

输入门 \(\mathbf{i}_t^{(l)}\) 控制候选细胞状态 \(\tilde{\mathbf{c}}_t^{(l)}\) 的写入比例。

两部分相加得到新的细胞状态 \(\mathbf{c}_t^{(l)}\)。这种**加法式记忆更新**（相比乘法更新）保证了梯度在时间轴上的稳定传播，是 LSTM 解决梯度消失问题的关键。

**公式 (3)-(7) 的内在联系：**

公式 (3)(4)(5)(6)(7) 共同构成 LSTM 细胞的核心计算流程：
1. 公式 (3) 的遗忘门决定"忘掉什么"；
2. 公式 (4) 的输入门与公式 (5) 的候选值共同决定"记住什么新信息"；
3. 公式 (7) 将两者综合，更新细胞状态；
4. 公式 (6) 的输出门控制"输出什么信息"。

---

#### 公式 (8)——隐状态输出（Hidden State Output）

\[
\mathbf{h}_t^{(l)} = \mathbf{o}_t^{(l)} \odot \tanh\left(\mathbf{c}_t^{(l)}\right)
\tag{8}
\]

**推导与解释：**

隐状态 \(\mathbf{h}_t^{(l)}\) 由两部分构成：
- \(\tanh(\mathbf{c}_t^{(l)})\)：将细胞状态非线性变换到 \([-1, 1]\) 范围；
- \(\mathbf{o}_t^{(l)}\)：输出门控制暴露程度，逐元素相乘。

**公式 (8) 与公式 (7) 的联系：**

公式 (7) 更新细胞内部记忆 \(\mathbf{c}_t^{(l)}\)，而公式 (8) 将这一记忆经过非线性变换和输出门过滤后，生成对外可见的隐状态 \(\mathbf{h}_t^{(l)}\)。\(\mathbf{h}_t^{(l)}\) 既是当前层向下一层传递的特征，也是递归传递到下一时间步的上下文。

**公式 (3)-(8) 与整体架构的联系：**

对于 \(m\) 层 LSTM 网络，第 \(l\) 层的输入为：

\[
\mathbf{x}_t^{(l)} = \begin{cases} \text{外部输入}(\text{如} a_g) & l = 1 \\ \mathbf{h}_t^{(l-1)} & l > 1 \end{cases}
\]

即上一层的隐状态输出作为下一层的输入。这些公式描述的单个 LSTM 单元机制，通过多层堆叠和全连接层，最终实现 \(a_g \rightarrow \mathbf{Z}\) 的序列到序列映射，为 PhyLSTM² 和 PhyLSTM³ 的架构提供基础模块。

### 2.3 PhyLSTM²——物理信息双 LSTM 网络

PhyLSTM² 由两个深度 LSTM 网络和一个张量微分器组成：
- **LSTM1**：预测状态空间变量 \(\mathbf{Z} = \{\mathbf{u}, \dot{\mathbf{u}}, \mathbf{r}\}^T\)，可训练参数为 \(\boldsymbol{\theta}_1\)；
- **LSTM2**：预测质量归一化恢复力 \(\mathbf{g}\)，可训练参数为 \(\boldsymbol{\theta}_2\)；
- **张量微分器**：通过中心差分计算 \(\mathbf{Z}\) 对时间的导数 \(\{\dot{\mathbf{z}}_1, \dot{\mathbf{z}}_2, \dot{\mathbf{z}}_3\}\)。

#### 公式 (9)——数据损失函数（Data Loss）

\[
\mathcal{J}_d(\boldsymbol{\theta}_1) = \sum_{i=1}^{n_d} \left\| \mathbf{Z}^{(i)}(\boldsymbol{\theta}_1) - \mathbf{Z}_d^{(i)} \right\|_2^2
\tag{9}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(n_d\) | 训练数据集数量（number of training datasets） |
| \(\mathbf{Z}^{(i)}(\boldsymbol{\theta}_1)\) | LSTM1 对第 \(i\) 个数据集预测的状态空间变量 |
| \(\mathbf{Z}_d^{(i)}\) | 第 \(i\) 个训练数据集的真实测量值 |
| \(\|\cdot\|_2^2\) | L2 范数的平方（均方误差意义下的距离） |

**详细解释：**

数据损失 \(\mathcal{J}_d\) 衡量 LSTM1 的预测值与已测量数据之间的误差。注意：
- 训练数据中 \(\mathbf{Z}_d = \{\mathbf{u}_d, \dot{\mathbf{u}}_d\}\)（位移和速度可测），而滞回参数 \(\mathbf{r}\) 不可测量；
- 这意味着 \(\mathbf{r}\) 的学习完全依赖物理约束损失来引导，单靠数据损失无法驱动对 \(\mathbf{r}\) 的有效预测。

公式 (9) 对应了**监督学习**部分，确保网络预测的可观测量与真实测量一致。

---

#### 公式 (10)——运动方程物理损失（EOM Physics Loss）

\[
\mathcal{J}_e(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2) = \sum_{i=1}^{n_c} \left\| \ddot{\mathbf{u}}^{(i)}(\boldsymbol{\theta}_1) + \mathbf{g}^{(i)}(\boldsymbol{\theta}_2) + \mathbf{\Gamma} a_g^{(i)} \right\|_2^2
\tag{10}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(n_c\) | 配点（collocation points）数量，可包括无标签数据点 |
| \(\ddot{\mathbf{u}}^{(i)}(\boldsymbol{\theta}_1)\) | 由 LSTM1 预测的加速度（通过张量微分器对 \(\dot{\mathbf{z}}_2\) 二次数值微分得到） |
| \(\mathbf{g}^{(i)}(\boldsymbol{\theta}_2)\) | 由 LSTM2 预测的质量归一化恢复力 |

**推导与解释：**

公式 (10) 直接来自公式 (2) 的物理约束。根据公式 (2)：

\[
\ddot{\mathbf{u}} + \mathbf{g} = -\mathbf{\Gamma} a_{g}
\]

移项得到残差形式：

\[
\ddot{\mathbf{u}} + \mathbf{g} + \mathbf{\Gamma} a_{g} = \mathbf{0}
\]

训练时要求网络预测的 \(\ddot{\mathbf{u}}\) 和 \(\mathbf{g}\) 满足这一约束，因此将残差的 L2 范数平方作为物理损失。

**与公式 (9) 的联系：**

\(\mathcal{J}_e\) 使用配点（可以是无标签数据），而 \(\mathcal{J}_d\) 仅使用有标签的训练数据。配点可以远多于训练数据，这使得物理约束能在整个时间域内有效施加，而不仅仅在有标签的时刻。这是"物理信息网络"的核心优势——用物理知识代替标签数据。

---

#### 公式 (11)——状态空间一致性损失（State Variable Consistency Loss）

\[
\mathcal{J}_g(\boldsymbol{\theta}_1) = \sum_{i=1}^{n_c} \left\| \dot{\mathbf{z}}_1^{(i)}(\boldsymbol{\theta}_1) - \mathbf{z}_2^{(i)}(\boldsymbol{\theta}_1) \right\|_2^2
\tag{11}
\]

**详细解释：**

状态空间变量被定义为 \(\mathbf{Z} = \{\mathbf{z}_1, \mathbf{z}_2, \mathbf{z}_3\}^T = \{\mathbf{u}, \dot{\mathbf{u}}, \mathbf{r}\}^T\)，因此存在以下恒等关系：

\[
\dot{\mathbf{z}}_1 = \dot{\mathbf{u}} = \mathbf{z}_2
\]

即位移的时间导数等于速度。这是一个**状态依赖性约束**（state dependency constraint）。

公式 (11) 要求张量微分器计算的 \(\dot{\mathbf{z}}_1\)（位移导数）与 LSTM1 直接预测的 \(\mathbf{z}_2\)（速度）保持一致。这一约束：
1. 强化了预测结果的物理自洽性（物理意义上位移的导数等于速度）；
2. 为 LSTM1 同时预测位移和速度提供了额外的监督信号；
3. 通过数值微分与网络输出之间的一致性约束，隐式增强了网络对时间积分关系的学习。

---

#### 公式 (12)——PhyLSTM² 优化问题

\[
\left\{\hat{\boldsymbol{\theta}}_1, \hat{\boldsymbol{\theta}}_2\right\} = \underset{\{\boldsymbol{\theta}_1, \boldsymbol{\theta}_2\}}{\arg\min} \; \mathcal{J}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2)
\tag{12}
\]

**详细解释：**

公式 (12) 描述了 PhyLSTM² 的训练目标：寻找最优参数 \(\hat{\boldsymbol{\theta}}_1\) 和 \(\hat{\boldsymbol{\theta}}_2\)，使总损失函数 \(\mathcal{J}\) 最小。

这是一个**多目标联合优化问题**，同时优化两个 LSTM 网络的所有权重和偏置。两个网络通过共享的物理损失 \(\mathcal{J}_e\) 相互耦合，形成"一体化"网络架构。

训练分两个阶段：
1. **预训练阶段**：使用 Adam 优化器（自适应动量估计），学习率 0.001，衰减率 0.0001，训练 \(10^4\) 个 epoch；
2. **精调阶段**：使用 L-BFGS（拟牛顿梯度优化算法）进一步精调，以提高收敛精度。

---

#### 公式 (13)——PhyLSTM² 总损失函数（Total Loss for PhyLSTM²）

\[
\mathcal{J}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2) = \mathcal{J}_d(\boldsymbol{\theta}_1) + \gamma \mathcal{J}_e(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2) + \mu \mathcal{J}_g(\boldsymbol{\theta}_1)
\tag{13}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\gamma\) | 运动方程物理损失的权重系数（用户定义，用于平衡数据损失与物理损失） |
| \(\mu\) | 状态一致性损失的权重系数（用户定义） |

**详细解释：**

公式 (13) 将三类损失线性加权求和，构成 PhyLSTM² 的**多目标综合损失函数**：

1. **\(\mathcal{J}_d\)（数据损失）**：驱动网络拟合可观测量（位移 \(\mathbf{u}\) 和速度 \(\dot{\mathbf{u}}\)），对应监督学习部分；

2. **\(\gamma \mathcal{J}_e\)（EOM 物理损失）**：施加运动方程约束，要求 LSTM1（预测加速度）和 LSTM2（预测恢复力）的输出满足公式 (2)；

3. **\(\mu \mathcal{J}_g\)（状态一致性损失）**：施加 \(\dot{\mathbf{z}}_1 = \mathbf{z}_2\) 约束，保证位移与速度的时间导数关系一致。

**权重系数的物理意义：**

- \(\gamma\) 和 \(\mu\) 控制物理知识与数据信息之间的相对重要性；
- 当可用数据极少时，增大 \(\gamma\) 和 \(\mu\) 可以让物理约束提供更多"监督信号"，防止过拟合；
- 论文中为简单起见取 \(\gamma = \mu = 1\)。

**公式 (9)-(13) 的内在联系总结：**

```
公式(9): 数据损失 J_d ─────────────────────────────────┐
公式(10): EOM 物理损失 J_e (来自公式(2))──────────────┤ → 公式(13) 总损失 J
公式(11): 状态一致性损失 J_g ──────────────────────────┘
公式(12): 训练优化目标 (最小化公式(13))
```

三个损失函数相互补充：数据损失保证预测精度，物理损失保证物理自洽性，状态一致性损失保证状态变量之间的内在关系。这三者共同约束网络在有限数据下学习到物理上合理的解。

### 2.4 PhyLSTM³——物理信息三 LSTM 网络

对于具有**速率相关滞回**（rate-dependent hysteresis）的系统，PhyLSTM² 的物理约束不够充分。论文进一步提出 PhyLSTM³，在 PhyLSTM² 基础上增加了对滞回参数微分方程的显式建模。

#### 公式 (14)——速率相关滞回系统的增广运动方程

\[
\left\{ \begin{array}{ll}
\ddot{\mathbf{u}} + \mathbf{g} = -\mathbf{\Gamma} a_g \\
\dot{\mathbf{r}} = f(\Phi)
\end{array} \right.
\tag{14}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\dot{\mathbf{r}}\) | 滞回位移参数 \(\mathbf{r}\) 的时间导数 |
| \(f(\cdot)\) | 描述滞回演化规律的非线性函数（未知，待学习） |
| \(\Phi\) | 系统变量库（library of system variables） |

**详细解释：**

公式 (14) 是公式 (2) 的**增广形式**，添加了滞回参数 \(\mathbf{r}\) 的微分方程约束。

对于 Bouc-Wen 模型，变量库取：
\[
\Phi = \{\Delta\dot{\mathbf{u}}, |\Delta\dot{\mathbf{u}}|, \mathbf{r}, |\mathbf{r}|^{n-1}, |\mathbf{r}|^n\}^T
\]

若先验知识未知，可简化为：
\[
\Phi = \{\Delta\dot{\mathbf{u}}, \mathbf{r}\}^T
\]

**与公式 (2) 的联系：**

公式 (14) 的第一方程与公式 (2) 完全相同，第二方程是新增加的物理约束，描述滞回参数随时间的演化规律。这一系统构成了**耦合常微分方程组**（ODE system），其中 \(\mathbf{u}\)（来自第一方程）和 \(\mathbf{r}\)（来自第二方程）相互耦合。

---

#### 公式 (15)——滞回损失函数（Hysteretic Loss for PhyLSTM³）

\[
\mathcal{J}_h(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3) = \sum_{i=1}^{n_c} \left\| \dot{\mathbf{r}}^{(i)}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3) - \dot{\mathbf{z}}_3^{(i)}(\boldsymbol{\theta}_1) \right\|_2^2
\tag{15}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\boldsymbol{\theta}_3\) | LSTM3 的可训练参数（权重和偏置） |
| \(\dot{\mathbf{r}}^{(i)}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3)\) | 由 LSTM3 预测的滞回参数时间导数 \(\dot{\mathbf{r}}\)，输入为 \(\Phi(\boldsymbol{\theta}_1)\) |
| \(\dot{\mathbf{z}}_3^{(i)}(\boldsymbol{\theta}_1)\) | 由张量微分器对 LSTM1 预测的 \(\mathbf{z}_3 = \mathbf{r}\) 进行数值微分得到的 \(\dot{\mathbf{r}}\) |

**详细解释：**

公式 (15) 的物理意义是：LSTM3 对 \(\dot{\mathbf{r}}\) 的显式预测（基于方程 \(\dot{\mathbf{r}} = f(\Phi)\)）应当与张量微分器对 LSTM1 输出 \(\mathbf{r}\) 进行数值微分的结果一致。

这一损失函数建立了以下两者之间的桥梁：
1. **LSTM3 的显式方程预测路径**：\(\Phi(\boldsymbol{\theta}_1) \rightarrow \text{LSTM3}(\boldsymbol{\theta}_3) \rightarrow \dot{\mathbf{r}}\)
2. **数值微分路径**：\(\mathbf{r}(\boldsymbol{\theta}_1) \rightarrow \text{张量微分器} \rightarrow \dot{\mathbf{r}}\)

两条路径的结果应趋于一致，即：
\[
\dot{\mathbf{z}}_3 - \dot{\mathbf{r}} \rightarrow 0
\]

**与公式 (11) 的类比：**

公式 (11) 要求 \(\dot{\mathbf{z}}_1 = \mathbf{z}_2\)（位移导数等于速度），而公式 (15) 要求两种不同路径计算的 \(\dot{\mathbf{r}}\) 一致。两者都是**自洽性约束**，确保网络的不同组件在物理上相互兼容。

#### 公式 (16)——PhyLSTM³ 优化问题与总损失

\[
\{\hat{\boldsymbol{\theta}}_1, \hat{\boldsymbol{\theta}}_2, \hat{\boldsymbol{\theta}}_3\} = \underset{\{\boldsymbol{\theta}_1, \boldsymbol{\theta}_2, \boldsymbol{\theta}_3\}}{\arg\min} \left[\mathcal{J}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2) + \eta \mathcal{J}_h(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3)\right]
\tag{16}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\eta\) | 滞回损失权重系数（用户定义，论文中取 \(\eta = 1\)） |
| \(\mathcal{J}(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2)\) | PhyLSTM² 的总损失（公式(13)） |
| \(\mathcal{J}_h(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3)\) | 新增加的滞回约束损失（公式(15)） |

**详细解释：**

公式 (16) 是 PhyLSTM³ 的完整优化目标，将 PhyLSTM² 的总损失（公式(13)）与新引入的滞回损失（公式(15)）加权求和：

\[
\text{总损失} = \underbrace{\mathcal{J}_d + \gamma \mathcal{J}_e + \mu \mathcal{J}_g}_{\text{PhyLSTM}^2 \text{ 损失 (公式13)}} + \underbrace{\eta \mathcal{J}_h}_{\text{新增滞回损失 (公式15)}}
\]

**与公式 (12)(13) 的联系：**

- 公式 (12) 是 PhyLSTM² 的优化问题，只涉及 \(\boldsymbol{\theta}_1\) 和 \(\boldsymbol{\theta}_2\)；
- 公式 (16) 将优化变量扩展到 \(\{\boldsymbol{\theta}_1, \boldsymbol{\theta}_2, \boldsymbol{\theta}_3\}\)，同时训练三个 LSTM 网络；
- \(\boldsymbol{\theta}_1\) 同时出现在 \(\mathcal{J}\) 和 \(\mathcal{J}_h\) 中，是 PhyLSTM³ 中三个子网络的"枢纽"，其通过梯度反传同时接受来自 LSTM2 和 LSTM3 的物理约束。

**PhyLSTM³ 物理约束体系：**

PhyLSTM³ 在 PhyLSTM² 的基础上，将物理约束从两个扩展到三个：

| 约束 | 数学表达 | 对应损失 |
|------|----------|----------|
| 状态空间一致性 | \(\dot{\mathbf{z}}_1 - \mathbf{z}_2 \rightarrow 0\) | \(\mathcal{J}_g\) 公式(11) |
| 运动方程 | \(\ddot{\mathbf{u}} + \mathbf{g} + \mathbf{\Gamma}a_g \rightarrow 0\) | \(\mathcal{J}_e\) 公式(10) |
| 滞回参数方程 | \(\dot{\mathbf{z}}_3 - \dot{\mathbf{r}} \rightarrow 0\) | \(\mathcal{J}_h\) 公式(15) |

## 第 4 章：Bouc-Wen 滞回模型验证

### 公式 (17)——Bouc-Wen 速率相关滞回方程

\[
\dot{r}_i = \Delta\dot{u}_i - \alpha_i |\Delta\dot{u}_i| |r_i|^{n_i - 1} r_i - \beta_i \Delta\dot{u}_i |r_i|^{n_i}
\tag{17}
\]

**符号说明：**

| 符号 | 含义 |
|------|------|
| \(\dot{r}_i\) | 第 \(i\) 个自由度滞回参数的时间导数 |
| \(\Delta\dot{u}_i\) | 第 \(i\) 个自由度的层间相对速度：\(\Delta\dot{u}_i = \dot{u}_i - \dot{u}_{i-1}\)（\(i \geq 2\)），\(\Delta\dot{u}_1 = \dot{u}_1\) |
| \(\alpha_i, \beta_i\) | Bouc-Wen 模型非线性参数，控制滞回形状 |
| \(n_i\) | Bouc-Wen 模型非线性指数，控制滞回从线性到饱和的过渡平滑度 |

**详细推导与解释：**

Bouc-Wen 模型是工程结构非线性滞回行为建模中最广泛使用的数学模型之一。公式 (17) 是其微分方程形式，具体物理意义分析如下：

**第一项：\(\Delta\dot{u}_i\)（线性项）**

代表弹性刚度贡献，表示在小变形（线弹性）阶段，滞回参数的变化速率与层间相对速度成正比。

**第二项：\(-\alpha_i |\Delta\dot{u}_i| |r_i|^{n_i-1} r_i\)（耗散项）**

- \(|\Delta\dot{u}_i|\)：绝对值确保了无论加载方向如何，耗散效果始终消耗能量；
- \(|r_i|^{n_i-1} r_i\)：这一组合同号，确保耗散项与 \(r_i\) 同号（从而使耗散力指向平衡点方向）；
- 当 \(|r_i|\) 较大时，该项主导，限制 \(r_i\) 的增长（饱和效应）。

**第三项：\(-\beta_i \Delta\dot{u}_i |r_i|^{n_i}\)（方向依赖耗散项）**

- 保留了 \(\Delta\dot{u}_i\) 的符号，使得加载和卸载阶段的行为有所区别；
- 结合 \(\alpha_i\) 项共同控制滞回环的形状（胖瘦、饱和程度等）。

**与公式 (14) 的联系：**

公式 (17) 是公式 (14) 中 \(\dot{\mathbf{r}} = f(\Phi)\) 的具体实例。在 PhyLSTM³ 中，LSTM3 被要求学习类似于公式 (17) 的映射关系，但无需提前知道 \(\alpha_i, \beta_i, n_i\) 等参数，而是从数据中自动辨识非线性滞回规律。

**数值求解过程（推导 \(r_i(t)\) 的演化）：**

给定初始条件 \(r_i(0) = 0\)，公式 (17) 可以用数值积分（如 Runge-Kutta 方法）求解：

\[
r_i(t + \Delta t) \approx r_i(t) + \dot{r}_i(t) \cdot \Delta t
\]

其中 \(\dot{r}_i(t)\) 由公式 (17) 给出，\(\Delta\dot{u}_i(t)\) 由结构运动方程（公式(14) 第一方程）确定。这体现了公式 (14) 所描述的耦合 ODE 系统的求解过程。

**Bouc-Wen 参数对滞回形状的影响：**

- \(n_i = 1\)：滞回过渡平滑；
- \(n_i \rightarrow \infty\)：滞回趋近于理想弹塑性（双线性）；
- \(\alpha_i + \beta_i > 0\)：系统具有能量耗散（对应物理上的有阻尼系统）；
- \(\alpha_i = \beta_i\)：滞回关于加卸载的对称性。

### 关键数学结构分析

**1. 双路径一致性约束（Two-path Consistency）**

PhyLSTM 框架的核心思想是引入两条计算路径并要求其结果一致：
- 路径1：LSTM 直接预测 → \(\mathbf{g}, \mathbf{r}, \dot{\mathbf{r}}\)
- 路径2：物理方程/数值微分 → 基于其他预测量计算得到的 \(\mathbf{g}, \mathbf{r}, \dot{\mathbf{r}}\)

损失函数 \(\mathcal{J}_e\)（公式10）、\(\mathcal{J}_g\)（公式11）、\(\mathcal{J}_h\)（公式15）分别对这三组双路径一致性进行约束。

**2. 多尺度监督信号**

- 有标签数据（\(n_d\) 个训练集）→ 数据损失 \(\mathcal{J}_d\)（公式9）
- 无标签配点（\(n_c\) 个配置点，通常 \(n_c \gg n_d\)）→ 物理损失 \(\mathcal{J}_e, \mathcal{J}_g, \mathcal{J}_h\)（公式10,11,15）

这使得网络能在数据稀缺的情况下，利用物理知识在更大范围内进行约束。

**3. 参数耦合结构**

\(\boldsymbol{\theta}_1\)（LSTM1）同时参与所有损失函数，是整个优化问题的核心：
\[
\mathcal{J}_{total} = \mathcal{J}_d(\boldsymbol{\theta}_1) + \gamma \mathcal{J}_e(\boldsymbol{\theta}_1, \boldsymbol{\theta}_2) + \mu \mathcal{J}_g(\boldsymbol{\theta}_1) + \eta \mathcal{J}_h(\boldsymbol{\theta}_1, \boldsymbol{\theta}_3)
\]

这种参数耦合使得 LSTM2 和 LSTM3 的物理知识通过反向传播传递到 LSTM1，从而间接引导 LSTM1 学习到物理上合理的状态变量时程，包括不可观测的滞回参数 \(\mathbf{r}\)。
